## 3. 專案架構

```
agri-web/

├── src/
│   ├── main.js                   # Vue 應用程式入口（路由配置）
│   ├── App.vue                   # 根組件
│   └── components/
│       ├── AgricHome.vue         # 首頁組件
│       ├── AgricEducation.vue    # 食農教育頁面
│       ├── IntegratorRegister.vue # 整合商登錄頁面
│       └── ReferencePlanning.vue  # 參考規劃頁面
└── assets/                       # 資源文件
    ├── SVG/                      # SVG 向量圖
    ├── 設備/                      # 設備圖片
    ├── 場域照片/                  # 場域安裝照片
    ├── 場域規劃/                  # 規劃參考圖片
    ├── 報導/                      # 新聞報導圖片
    └── 規格圖片/                  # 設備規格圖片
```

## 4. 頁面路由

| 路由路徑 | 組件 | 頁面名稱 | 功能說明 |
|---------|------|--------|---------|
| `/` 或 `/home` | AgricHome | 首頁 | 產品展示、場域實績、技術亮點、新聞報導 |
| `/education` | AgricEducation | 食農教育 | 食農教育課程、SDGs 結合、實踐案例 |
| `/integrator-register` | IntegratorRegister | 整合商登錄 | 整合商專區、設備規格、場域參考、報價 |
| `/reference-planning` | ReferencePlanning | 參考規劃 | 多種場域規劃方案、設備搭配建議 |

## 5. 組件詳細說明

### 5.1 AgricHome.vue（首頁）

**程式碼行數**: ~1016 行

**主要區塊**:
1. **Header 導航** - 固定頂部，支持手機菜單
2. **Hero Section** - 背景圖 + 波浪 SVG + 號召行動按鈕
3. **全台安裝場域** - 台灣地圖標記 + 點擊展示場域照片
   - 桃城蒔菜（6張照片）
   - 富春山農場（3張照片）
   - 儒園鮮果（2張照片）
4. **農研技術核心科技** - 4 個技術特點卡片
5. **新一代樂農小主機** - 產品介紹
6. **主機介紹** - 3 款產品輪播
7. **智慧管理好輕鬆** - 3 個功能卡片
8. **串接氣象站** - 儀表板圖片展示
9. **場域控制一目瞭然** - 儀表板詳細展示
10. **我們可以控制什麼** - 12 個控制項目圖標
11. **誰適合使用樂農** - 3 類用戶介紹
12. **相關報導** - 8 個新聞/影片卡片
13. **Footer** - 聯絡資訊、快速連結

**互動功能**:
- Intersection Observer 滾動動畫
- 觸控滑動產品輪播（手機版）
- 農場彈出視窗
- 信箱複製功能

## 8. 開發命令

```bash
# 安裝依賴

# 生產構建
npm run build

# 預覽構建結果
npm run preview

# 部署到 GitHub Pages
npm run deploy
```
