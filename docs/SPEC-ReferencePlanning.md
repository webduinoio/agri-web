### 4.1 溫室環控｜定時灌溉

**規劃參考**:

|------|------|
| 系統說明 | 環境數據收集、定時灌溉控制 |

  showPopup: false,         // 彈出視窗開關

## 6. 互動功能

### 6.1 方案卡片點擊

- 點擊卡片開啟彈出視窗
- 顯示對應方案的詳細內容
- 禁止背景滾動

### 6.2 彈出視窗

**內容結構**:
1. 標題與情境圖
2. 兩欄佈局：規劃參考 + 設備參考
3. 桃改場專家積木展示

**功能**:
- 點擊背景關閉
- 點擊關閉按鈕關閉
- 內容區可捲動

### 6.3 圖片放大

- 點擊專家積木圖片可放大查看
- 黑色半透明背景
- 點擊背景或關閉按鈕關閉

## 9. 方法列表

| 方法名 | 功能 |
|--------|------|
| `toggleMobileMenu()` | 切換手機選單 |
| `openPopup(type)` | 開啟指定類型的彈出視窗 |
| `closePopup()` | 關閉彈出視窗 |
| `handleImageError(event)` | 處理圖片載入失敗 |
| `openImageModal(src, alt)` | 開啟圖片放大模態框 |
| `closeImageModal()` | 關閉圖片放大模態框 |
